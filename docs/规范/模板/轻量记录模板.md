# 轻量记录模板

适用于原型、一次性验证、小范围内部工具，或紧急修复的临时记录。不替代中等和重大真实业务需求在**已启用 OpenSpec 时**的 OpenSpec 记录；未启用 OpenSpec 时，中等及以上需求须使用外部需求系统或经用户确认的其他可追溯载体。

复制到任务记录、issue、PR 描述或交付说明中使用：

```text
变更范围：
- ...

验证：
- ...

未验证/剩余风险：
- ...

是否需正式需求记录 / OpenSpec：
- 否，原因：...（若项目已启用 OpenSpec 且属于中等及以上需求，应填「是」并注明 change 路径）
```

PR 场景可同时填写 `.github/pull_request_template.md` 中的对应字段。
