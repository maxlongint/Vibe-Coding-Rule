# 规范总览

> **模板说明：** 本文件供复制到**业务项目**的 `docs/规范/` 使用。在本规则仓库中作为规范目录索引。

本目录存放**项目级**长期规则与可选治理细则，例如 UI 规范、API 约定、需求治理、AI 工具链分工等。

## 与 AGENTS.md 的分工

| 位置 | 放什么 | 项目新增时 |
| --- | --- | --- |
| `AGENTS.md` 第 1–4 节 | 工程守则、注释、编码、提交与发布（跨项目强制） | **不修改**，随规则包升级 |
| `AGENTS.md`「规则包附带文档」 | 仅列出规则包固定附带的路径 | **不修改** |
| 本文件 `规范总览.md` | 项目特有规范索引 | **只改本目录** |
| `docs/规范/` 其他文件 | 需求治理、AI 工具链、UI/API 等项目约定 | **只改本目录** |

无任务相关项目规范时，AI 按 `AGENTS.md` 执行，无需预建空文件。
