# 规范总览

> **模板说明：** 本文件供复制到**业务项目**的 `docs/规范/` 使用。在本规则仓库中仅作示例索引，不代表本仓库自身的项目规范。

本目录存放**业务项目特有**的长期规则，例如 UI 规范、API 约定、业务工作流、领域术语等。

## 与 AGENTS.md 的分工

| 位置 | 放什么 | 项目新增时 |
| --- | --- | --- |
| `AGENTS.md` 第 4–7 节 | 工程守则、注释、编码、提交与发布（跨项目强制） | 不修改，随规则包升级 |
| `docs/规范/` | 本项目特有的业务与工程约定 | 只增改本目录内文件 |

无任务相关项目规范时，AI 按 `AGENTS.md` 执行，无需预建空文件。

## 索引（由项目自行维护）

复制到业务项目后，若尚未添加特有规范，可在此说明；新增时在此列出，例如：

```text
- UI规范.md
- API约定.md
```
