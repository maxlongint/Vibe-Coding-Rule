# 项目级技能包索引

本目录存放 Vibe Coding 规则包提供的项目级技能。技能包用于补充 `AGENTS.md` 的执行方法，不替代使用方项目自己的规范文档、OpenSpec 需求记录或业务规则。

## 技能列表

| 技能 | 适用场景 | 触发方式 |
| --- | --- | --- |
| `executing-complex-requirements` | 复杂、高风险或跨模块产品需求，尤其是会改变行为、接口、数据、权限、状态流转、用户路径、验收标准、发布或回滚的变更 | 可由任务类型自动触发，也可由用户点名 |
| `updating-requirement-changes` | 已有需求的补充、范围变化、验收调整、需求正文更新和 OpenSpec change 同步 | 仅在用户明确点名或明确要求使用需求变化同步技能时触发 |

## 使用边界

- 本仓库是技能包和通用规则包，不是业务项目本身；不要求在本仓库初始化 `.codegraph/` 或 `openspec/`。
- `AGENTS.md` 是固定基线，不承载不同项目会变化的技术栈、业务规范或命令细节。
- 使用方项目的差异化规则应放在项目 README、`docs/规范/`、OpenSpec、外部需求系统或项目自己的技能包中。
- 根目录的 `SSO 单点登录复杂需求执行步骤示例.md` 是保留的教学示例，不是本仓库自身的 OpenSpec 需求记录。
